# Git 사용법

## Git 시작하기에 앞서

1. Git이란?

    **소스형상 관리 시스템**  

    * 여러 사람들이 협업하여 프로젝트를 진행하는 경우  
소스코드나 각종 **파일의 공유**, **버전닝(Versioning)**, **릴리즈(Release)** 를 관리하기 위해  
사용하는 시스템  

    * 소스형상 관리 시스템의 분류
        - 클라이언트-서버 모델
        - 분산 모델
    * Git은 **분산모델** 에 해당
    * 분산모델이란, 전체 저장소에 대해 개발자가 각자의 **Local Repository** 를 가지고 있으며,  
    전체 저장소에 대한 사본을 유지
    * 전체 저장소는 일반적으로 **Github** 를 이용
